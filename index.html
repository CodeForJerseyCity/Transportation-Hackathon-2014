  • About
    The Hackathon is organized by OpenJC and CodeForNewark
    Open JC
    Mission: Open Government Data + Solve civic problems with technology
    Jersey City, New Jersey, USA
    Open JC is a hands on active community group collaborating with the Jersey City Government to build open data, open source and open government in Jersey City. We believe in public sector innovation and the value of digital-government projects. Come join other developers, designers, data geeks, civic leaders, organizers, and idea-makers to volunteer and collaborate.
    Code For Newark
    Mission: To improve our communities and the governments that serve them
    Newark, New Jersey, USA
    Code For Newark is a Code for America Brigade, which means we're volunteers that contribute our talents toward improving the way our local governments and community organizations use the web. Brigade is about action and you're here because you are someone who takes action.

  • Tools
    Suggested list of mapping tools
    Google Maps
    Google Maps is a map service that you view in your web browser
    https://developers.google.com/maps/documentation/
     
    Google Maps has a wide array of APIs that let you embed the robust functionality and everyday usefulness of Google Maps into your own website and applications, and overlay your own data on top of them.

    1) Example using GTFS:  http://simcity.vasile.ch/geneva/

    2) Example of mapping 'biking wisdom' (NYT built a form that popped on the map, and then plugged that database in with the Maps API, Google Form > Google Spreadsheet > Maps API map):  http://www.nytimes.com/newsgraphics/2013/06/07/bicycle-routes/

    3) Overlay GeoJSON on a Google map with new Data API:  https://developers.google.com/maps/documentation/javascript/datalayer

    4) Google Geo Media Lab:  https://sites.google.com/site/geomedialab/

    MapBox
    MapBox is an open source mapping platform
    https://www.mapbox.com
    MapBox provides tools to create maps using open source software and open data.
    TileMill by MapBox
    Create beautiful interactive maps
    https://www.mapbox.com/tilemill/
    Whether you're a journalist, web designer, researcher, or seasoned cartographer, TileMill is the design studio to create stunning interactive maps.
    QGIS
    A Free and Open Source Geographic Information System
    http://www.qgis.org/
    QGIS is a user friendly Open Source Geographic Information System (GIS) licensed under the GNU General Public License. QGIS is an official project of the Open Source Geospatial Foundation (OSGeo). It runs on Linux, Unix, Mac OSX, Windows and Android and supports numerous vector, raster, and database formats and functionalities.
     
    There is an open source web clinet that renerst the QGIS maps for web apps:  https://github.com/qgis/QGIS-Web-Client

    GRASS GIS
    GRASS GIS is a free, open source geographical information system
    http://grass.osgeo.org/
    GRASS GIS, commonly referred to as GRASS (Geographic Resources Analysis Support System), is a free and open source Geographic Information System (GIS) software suite used for geospatial data management and analysis, image processing, graphics and maps production, spatial modeling, and visualization. GRASS GIS is currently used in academic and commercial settings around the world, as well as by many governmental agencies and environmental consulting companies. It is a founding member of the Open Source Geospatial Foundation (OSGeo).
    CartoDB
    A cloud-based solution for all your mapping needs
    http://cartodb.com/
    Create dynamic maps, analyze and build location-aware and geospatial applications with your data using the power using the power of PostGIS in the cloud.
    GTFS
    GTFS Transportation Standard
    https://developers.google.com/transit/gtfs/
    The General Transit Feed Specification (GTFS) defines a common format for public transportation schedules and associated geographic information. GTFS "feeds" allow public transit agencies to publish their transit data and developers to write applications that consume that data in an interoperable way.
     
    GTFS specification:  https://developers.google.com/transit/gtfs/reference
    Open Street Map
    OpenStreetMap is open data and open source software
    http://www.openstreetmap.org/relation/170953
    OpenStreetMap is built by a community of mappers that contribute and maintain data about roads, trails, cafés, railway stations, and much more, all over the world.
     
    http://developer.mapquest.com/web/products/open/tools/guide
     
    kdmcBerkeley maps tutorials
    Mapping resources and tutorials
    http://multimedia.journalism.berkeley.edu/tutorials/cat/maps
    KDMC, Knight Digital Media Center at UC Berkeley Graduate School of Journalism, produces a wealth of digital media tutorials to support our training sessions and classes. While the focus of some tutorials is on technology and journalism, most are general enough to be of use to anyone.
    Natural Earth Data
    Free vector and raster map data
    http://www.naturalearthdata.com/
    Natural Earth is a public domain map dataset available at 1:10m, 1:50m, and 1:110 million scales. Featuring tightly integrated vector and raster data, with Natural Earth you can make a variety of visually pleasing, well-crafted maps with cartography or GIS software.
